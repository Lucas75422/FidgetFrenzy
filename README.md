# FidgetFrenzy Store

## Project Description
FidgetFrenzy is an online store dedicated to providing a wide variety of fidget toys and tools. Our goal is to help individuals of all ages discover the benefits of fidgeting, whether for focus, stress relief, or sensory stimulation. From fidget spinners to stress balls, we have something for everyone.

## Setup Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/Lucas75422/FidgetFrenzy.git
   cd FidgetFrenzy
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm start
   ```
4. Open your browser and go to `http://localhost:3000`
